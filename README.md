# Combat HARD Core

Unityで制作した1作目のゲームを公開しているリポジトリです。

# DEMO

#### ゲームジャンル

3Dシューティング

#### ルール

・目標の場所までたどり着くとクリア<br>
・道中、敵からの攻撃や障害物に体力を削り切られるとゲームオーバー

#### 操作方法

・上：　　W　　　・左右旋回：　A,D<br>
・下：　　S　・攻撃：　Space<br>

#### プレイ映像

https://youtu.be/-gUdlKJysrI

* DOTween

# Usage

Unityでビルドして実行して下さい
